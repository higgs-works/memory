# Memory

> Human-Like Memory System for AI

AI를 위한 인간형 기억 시스템. 검색이 아닌, 자동 표면화.

## 프로젝트 개요

**Human-Like Memory System**은 검색 기반이 아닌, 연상 활성화(Spreading Activation)와 자동 표면화를 통해 관련 기억이 컨텍스트에 맞게 자동으로 떠오르는 시스템이다.

## 핵심 아이디어

기존 AI 메모리 서비스는 결국 **검색 엔진**이다. 하지만 인간의 기억은 검색하지 않는다. 관련 상황에서 **자동으로 떠오른다**.

```
기존: "JWT 관련 기억 검색해줘" → 결과
우리: (인증 얘기 중) → "아, 그때 JWT 선택한 이유가..." 자동 표면화
```

## 핵심 원칙

1. **검색 없는 기억**: 명시적 검색 없이 자동 표면화
2. **연상 네트워크**: 기억 간 연결을 통한 활성화 전파
3. **동적 강도**: 사용 패턴에 따른 기억 강도 변화
4. **에피소드 중심**: 청크가 아닌 경험 단위 저장

## 기존 서비스와의 차별점

| 기존 (Supermemory 등) | 우리 |
|---------------------|-----|
| 검색 기반 | 자동 표면화 |
| 벡터 유사도 | 연상 네트워크 |
| 정적 저장 | 동적 강도 변화 |
| 청크 단위 | 에피소드 단위 |
